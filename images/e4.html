Raspberry Pi® 3 Model B

The Raspberry Pi ® 3B+ is a powerful successor to the extremely popular credit card-sized single-board computer. This 3rd-generation Raspberry comes with a high-performance ARM Cortex A53 4x 1.4 GHz quad core processor. By this upgrade gives the Pi ® 3 B+ in certain applications significantly more power. For quick and easy networking are Wi-Fi and Bluetooth® Low Energy and GB-LAN onBoard. The layout and the connections of the Raspberry Pi ® 3 B+ are identical to the Raspberry Pi ® 3 Model B and Pi ® B+.

The changes in the overview:
The Raspberry 3B+ is since March 2018, the successor of the successful Raspberry 3B. It is in the base identical to its predecessor and is fully compatible with all accessory components. So you can continue to use all the housing and expansion boards. The following improvements/changes were made in the 3B+:
1) The CPU is remained the same but was in the performance.
2) The LAN interface works 3x faster than with the Raspberry 3B.
3) The WLAN has been improved: AC
4) The 3B+ now supports Bluetooth ® 4.2.
5) Optional Power over Ethernet (via separate has module)

Full compatibility with existing accessories
The other in the Raspberry Pi ® 3 B+ installed components, connections as well as the form factor nothing has changed. This is a complete compatibility with accessories for previous versions.
